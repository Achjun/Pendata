# Pertemuan 2


## MEMAHAMI DATA

Pada pertemuan 2 membahas topik tentang *memahami data*.

Sebelum melakukan analisis data, kita perlu memahami terlebih dahulu tujuan dari analisis tersebut.  
Kita juga perlu mengetahui data apa saja yang dibutuhkan untuk melakukan analisis.

Pemahaman data sangat diperlukan untuk:
- Mendeteksi bagian yang menarik dari data
- Menemukan pola dalam data
- Membangun hipotesis awal
- Menghindari kesalahan dalam analisis  


## Pentingnya Memahami Data

Memahami data sangat penting karena:

- Membantu memilih teknik data mining yang tepat  
- Meningkatkan akurasi model prediksi  
- Menghindari kesalahan interpretasi  
- Memastikan hasil yang dapat diandalkan dan dapat ditindaklanjuti
